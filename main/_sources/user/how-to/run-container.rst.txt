Run in a container
==================

Pre-built containers with EPICSPyClientPerformance and its dependencies already
installed are available on `Github Container Registry
<https://ghcr.io/ajgdls/EPICSPyClientPerformance>`_.

Starting the container
----------------------

To pull the container from github container registry and run::

    $ docker run ghcr.io/ajgdls/EPICSPyClientPerformance:main --version

To get a released version, use a numbered release instead of ``main``.
